Publish as a live, swipeable artifact

This downloads a standalone viewer of your carousel — present-mode only, no editing controls, just the swipeable artifact. Host it free on GitHub Pages and share the link; anyone who opens it swipes through your live carousel.

Then publish it free in 4 steps:
1. Go to github.com → New repository → name it (e.g. my-carousel) → Public → Create.
2. Click uploading an existing file → drag in the downloaded index.html → Commit.
3. Repo Settings → Pages → Source: main branch → Save.
4. Wait ~1 min. Your live link appears: username.github.io/my-carousel
Share that link anywhere. It opens your live swipeable artifact.
